# 쇼츠 스크립트 구조와 길이

[[creator]]가 스크립트를 쓸 때의 골격. **검증된 사실과 미검증 통념을 구분**해 둔다. 정량 구조는 딥리서치(231만 틱톡 게시물·500+ 바이럴 분석·한국언어청각임상학회지)로 검증됨 — `/squeeze-report` 처리(2026-06-07).

## 검증된 골격: Hook → Body → CTA

- **Hook (첫 ~3초):** 시청/스와이프를 가르는 결정 구간. 질문형·카운트다운·결과 먼저보기 중 하나. → [[바이럴 훅 3초 법칙]], [[쇼츠 알고리즘 우대 신호 2026]]
- **Body:** 완주율이 1순위 신호이므로 **끝까지 보게** 설계. 늘어지면 이탈.
- **CTA:** 마지막 3초도 효과(+31%) — 구독/다음 영상 유도를 끝에 명확히.

## 길이·발화량 (검증된 사실)

- YouTube Shorts: **최대 3분** 가능(2024.10~). 단 길다고 알고리즘이 우대하진 않음 — 완주율이 핵심.
- TikTok 수익화: **5초+ = Qualified View**.
- **누적 보상 최적 길이: 31~60초** — 리텐션 60% 방어 시 절대 시청시간(총 체류)이 극단적 짧은 영상보다 큼. 단순 완주율 100%만 노리는 15초 미만보다 우대. (출처: opus.pro 리텐션 데이터)
- **발화 속도(검증):** 영어 숏폼 170~200 WPM. 단 10~15초 주기 쉼표 삽입 시 정보 기억률 23~27% 상승(에든버러대). 극단적 빠른 템포는 인지 과부하로 역효과(DTU·SDSU).

## ⭐ 한국어 음절(SPM) 기준 — WPM 1:1 전이 금지 (검증된 사실)

영어 "단어(Word)" 기준을 한국어에 그대로 못 쓴다. 한국어는 교착어라 **음절(SPM, Syllables Per Minute)**이 정확한 밀도 기준. (출처: 한국언어청각임상학회지 TV뉴스 아나운서 발화 연구)

- **한국어 정보전달 최적 발화: ~356 SPM**(아나운서). 숏폼 텐션은 1.2배인 **400~420 SPM(초당 6.5~7음절)**까지 허용.
- **60초 스크립트 ≈ 350~400음절** 내외에서 인지 부하 안정. **600음절 초과 시 과부하**.
- **⭐ 3초 훅 = 최대 18~21음절** (초당 7음절 가정). 예: "당신이 반드시 알아야 할 단 하나의 마케팅 비법"(19음절)≈2.5~3초. 훅이 21음절 넘으면 3초 안에 다 못 읽힘.
- **전이 규칙:** 시간 비율(훅5/본문50/CTA5초)은 인간 주의력 한계라 언어 무관 전이됨. 하지만 **발화량은 단어수 아닌 음절수로 계산**해야 함. → [[훅 구체성 역U자 법칙]]의 한국어 적용과 정합.

## 구간 배분 (검증된 시간 비율)

500+ 바이럴 영상 분석상 시간 비율이 수렴(reddit 500개 분석·sovran.ai):

| 길이 | 훅 / 본문 / CTA |
| :-- | :-- |
| 15초 | 2 / 11 / 2초 |
| 30초 | 3 / 24 / 3초 |
| 60초 | **5 / 50 / 5초** (역피라미드) |
| 90초 | 5 / 80 / 5초 |

- **플랫폼별 훅 시간:** 메타/릴스 1~3초, 틱톡 1~2초, 유튜브 쇼츠 3~5초.
- **정보 사다리(Information Ladder):** 본문에서 **10~15초마다** 새 정보·반전·시각 충격을 던져 중간 이탈(Sag) 방어. 바이럴 32.7%가 채택.
- **가치 우선(Value-First):** 결론을 끝까지 숨기지 말 것(최악 성과). 초반 15초에 핵심 결과/노하우 투척 → 이후 이유·방법 해설. 바이럴 19.2%.

## 이탈 방어 (검증된 사실)

- **패턴 인터럽트:** 3~5초마다 시점 변경·텍스트 오버레이·B-roll 전환 → 완주율 **41%→58%**. talking-head 정적 영상은 41%에 그침.
- **자막:** 화자 싱크 자막만으로도 리텐션 **+12%**.
- **심리스 루프:** 마지막 문장↔첫 문장 자연 연결 → 재시청률 +18.7%·도달 +23.6%(페이스북 릴스). 단 **5초 미만 무한루프 꼼수는 금지** — 유튜브 2021.12 업데이트로 루핑 스팸 트래픽 페널티. 정보·감정 밀도 있는 의미있는 재시청만 유효.
- **리텐션 목표선: 75% 방어.** 80%+ 완주 영상은 글로벌 추천피드로 노출 폭발. (플랫폼 평균: 틱톡 78%·쇼츠 73%·릴스 65%)

## 해시태그 전략 (검증된 사실)

Metricool 231만 틱톡 게시물 분석(2026):

- **개수: 1~5개** (1~2개 최적, 5개 상한). "Less is more". 10~15개 채우기는 알고리즘 분류 혼란 → 역효과.
- 해시태그 사용 시 조회수 **+5%**, 상호작용 **+10%**. (해시태그 무용론은 틀림 — 트래픽 +114% 성장)
- **#fyp·#viral 등 광범위 태그 배제** — 알고리즘에 단서 0. **본문과 정확히 일치하는 니치 태그**(예: 채널 도메인 키워드)를 캡션(설명란)에 배치. 조회 70~73%가 추천피드(FYP)에서 발생하므로 타겟 군집 식별이 핵심.

## ⚠ 여전히 미검증 (단정 금지)

- 영어 180WPM 리텐션 곡선이 한국어 400SPM에서 동일 이탈 곡선을 그리는지 대조검증 — **시간 구조는 호환, 발화량 백분율 이탈은 미검증** → 운영 데이터 후 [[훅 구체성 역U자 법칙]]과 함께 재평가
- 주제(코미디/정보성/Vlog)별 WPM 초과 임계치 세분류
- 플랫폼 간 해시태그 가중치 정확한 상수

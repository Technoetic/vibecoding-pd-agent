# /day — 기획 일지 캡처

입력: **$ARGUMENTS**

1. 시스템 날짜(`YYYY-MM-DD`)를 확인하고 `30-journal/YYYY-MM-DD.md`가 있는지 확인.
2. 없으면 [[frontmatter]] 스키마(`type: journal`)로 새 일지 생성.
3. 입력 텍스트를 분석해 프로젝트명·인물·핵심 개념·키워드를 식별하고, **그 명사들에 위키링크 `[[ ]]`를 공격적으로 씌워** 일지 본문에 추가.
4. 새로 연결된 개념이 `20-knowledge/`에 없으면 고아 링크로 남겨 둔다(나중에 `/ingest`로 채움).

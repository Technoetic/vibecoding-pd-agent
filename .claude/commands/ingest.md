# /ingest — 데이터 섭취 및 원자적 분해

대상: **$ARGUMENTS** (없으면 `10-inbox/`의 미처리 파일을 순회)

1. **탐색:** `00-meta/index.md`·`log.md`를 읽어 기존 지식 파악.
2. **소화:** 대상 소스를 읽고 가장 중요한 3~5개 핵심 개념/결론 추출.
3. **분해·연결:** 각 개념을 별도 원자적 노트로 분할해 `20-knowledge/`의 적절한 하위(`hooks/` `scripts/` `trends/`)에 생성.
   - [[frontmatter]] 스키마 준수. `related`에 출처와 연관 노트를 위키링크로 기재(이중 따옴표).
   - 본문 핵심 명사는 `[[ ]]`로 공격적 링킹.
4. **등록:** `00-meta/index.md`에 새 노트 링크 추가.
5. **격리:** 처리된 원본을 `10-inbox/_processed/`로 이동(중복 스캔 방지).
6. **기록:** `00-meta/log.md` 최상단에 1줄 요약.

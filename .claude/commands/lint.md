# /lint — 볼트 무결성 검증

`20-knowledge/`·`30-journal/`·`50-projects/`·`00-meta/`를 순회하며(격리 `_` 폴더·`90-assets`·`_가이드` 제외) 다음을 진단해 보고서로 출력한다:

1. **프론트매터 누락:** [[frontmatter]] 필수 필드가 없는 `.md`.
2. **YAML 위반:** 프론트매터 내 따옴표 없는 `[[ ]]` (Dataview 붕괴 위험).
3. **데드 링크:** `[[ ]]` 대상 파일이 없는 경우 (단, 의도된 고아 링크와 구분).
4. **고아 노트:** 어떤 노트에서도 링크되지 않은 문서.
5. **경로 위반:** `20-knowledge/` 하위에 새로 생긴 폴더(평면 구조 위반).
6. **도메인 불일치:** 물리 경로와 프론트매터 `type`/`domain` 충돌.

발견된 문제는 심각도순으로 나열하고, 자동 수정 가능한 항목은 사용자 확인 후 일괄 수정한다. 모순 노트는 `> [!contradiction]` 콜아웃으로 표시.
